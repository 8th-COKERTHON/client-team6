|   1. 회원가입 |  |  |  | 사용자가 이메일로 회원가입

|  | 1.1 온보딩 - 사건 5개 등록 |  |  | 그동안 자신이 겪은 안좋은 일 5개를 연달아 적도록 유도
  • 사건 제목 : AI가 자동 생성, 사용자가 수정 가능 (필수항목)
  • 사건 내용 : 사용자가 자율적으로 입력 (선택 항목)
  • 날짜 : 입력 날짜로 디폴트 값 들어가 있음, 사용자가 수정 가능 - 날짜 = 해당 사건이 일어난 날짜 (필수 항목)
      ◦ (E1) 기억이 잘 안나는 옛날 일일 경우, 월만 입력 가능
      ◦ (E2) 5개 적기 힘든 경우 → 언제든 나갈 수 있는 버튼 제공 (5개를 모두 안적은 경우, 향후 대진에서 적은 개수만큼만 매칭 진행) |
|  |  | 1.1.1 온보딩 배치전 진행 |  | 5개의 사건에 대한 순위 결정전 배치 진행
  • 리그전으로 진행 총 10판
  • 매치 순서 = 등록 순서 그대로
  • 리그전에서 1등을 한 사건 = 올타임 챔피언 |

  • Monthly Royal Rumble 매칭 링 입장 푸시 알림 (월말)
  • 앱 이벤트가 있는 경우, 푸시 알림 |
|  | 1.2 오늘의 사건 등록 |  |  | 사용자가 기분이 안좋은 일을 겪었을 때, 오늘의 사건 등록을 통해 자신의 일을 적을 수 있음 |

  • 사건 내용 : 사용자가 자율적으로 입력 (선택 항목)
  • 날짜 : 입력 날짜로 디폴트 값 들어가 있음, 사용자가 수정 가능 (필수 항목) |

질문 : 현재 기준, 더 힘든 상황은?
  • 카드 형식으로 대결 화면이 나타나고, 내용보기를 누르면 카드가 뒤집히면서 내용이 나타남 (이때 카드에서 나타나는 최대 글자수 : 60자)
      ◦ (E1) 60자 넘게 내용을 입력한 경우 하단의 내용보기를 누르면 팝업에 전체 내용이 뜨게 됨 |

매칭 리스트 |  |  | 현재 사용자가 진행할 수 있는 매칭리스트가 나타남 |
|  | 3.2 앞으로 진행가능한 매칭 리스트 |  |  | 현재 진행중이지는 않지만, 앞으로 진행 가능한 매칭리스트가 나타남 |
|  |  | 3.2.1 월간 챔피언 리그 |  | 상위 10개의 랭커들에 대해 대결 진행
  • 이긴 카드가 계속해서 다른 카드와 대결을 진행하는 구조
  • 대결일 : 매월 말일
  • 대결 진행 이후, 결과 화면 나타남 |
|  |  | 3.2.2 연간 챔피언 리그 |  |  • 월간챔피언 리그를 진행한 바로 다음에 연간 챔피언 경쟁이 일어남
  • 대결일 : 매월 말일
  • 대결 진행 이후, 결과 화면 나타남 |
| 4. 랭킹 | 4.1 월간 챔피언 |  |  | 월간 챔피언이 나타남
  • 초대 월간 챔피언 : 처음 달에서 Monthly Royal Rumble를 진행했을 때 1등을 한 사건 |
|  | 4.2 연간 챔피언 |  |  | 연간 챔피언이 나타남
  • 초대 연간 챔피언 = 초대 월간 챔피언
  • 그 다음부터는 연간 챔피언 vs 해당 달의 월간 챔피언 → 연간 챔피언을 임명 |
|  | 4.3 올타임 챔피언 |  |  | 올타인 챔피언(=전체 기간 중에서 가장 힘든일)이 나타남 |
|  | 4.3 검색바 |  |  | 제목/내용기반 검색 기능 |
|  | 4.4. 랭킹 리스트 |  |  | 그간 랭킹 리스트가 나타남 |
|  |  | 4.4.1 상세 내용 |  | 해당 리스트를 클릭하면 사용자가 이전에 적은 내용이 나타남 |
| 5. 기록실 | 5.1 검색바 |  |  | 제목/내용기반 검색 기능 |
|  | 5.2 역대 매칭 기록 |  |  | 역대 매칭기록들이 리스트로 나타남
  • 승/패도 나타남 |
